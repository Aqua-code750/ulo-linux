# Ulo Linux v1.0.0 - "Genesis" Release

Welcome to the very first release of **Ulo Linux**, the Agentic Operating System built from the ground up for seamless QEMU virtualization, AI automation, and premium aesthetics.

## What's Included in this ISO:
- **Custom Kernel (`linux-ulo`)**: Based on Linux 6.6 LTS, heavily debloated, and injected with built-in `virtio` drivers for zero-latency VM booting.
- **UloShell (AGS + Hyprland)**: A completely bespoke UI written in JavaScript and CSS, featuring a macOS-style dock, rounded floating windows, and glassmorphism.
- **Ulo Agent**: The system-wide AI daemon integrated into the bash terminal, file explorer, and global search.
- **Ulo First-Party Apps**:
  - **Ulo Browser**: Firefox heavily customized for privacy and UI integration.
  - **Ulo Files & Store**: Blazing fast Rust/Tauri applications for managing your system.

## How to Run:
If you are on Windows, simply place `ulo-linux-release.iso` in the same folder as `launch_qemu.bat` and double click the batch file to instantly boot into Ulo Linux using QEMU with full virtio hardware acceleration.
